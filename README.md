# Toy ReLU Project

This project implements a simple neural network with ReLU activation function for educational purposes.

## Project Structure

- `train.py` - Main training script
- `models/` - Directory for saved model weights

## Requirements

- Python 3.x
- PyTorch
- NumPy
- tqdm

## Usage

To train the model, run:
```bash
python train.py
```
